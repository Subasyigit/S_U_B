# S_U_B
Merhaba! Ben size Amazon'da ne kadar harcadığınızı analiz etmenizi sağlamak için bir Python kodu yazdım. Bu, Amazon hesabınızın işlem geçmişine erişmeniz ve bu verileri toplamanız, daha sonra da istediğiniz gibi analiz etmeniz gerektiği anlamına gelir.

Burada bir örnek Python kodu vermek istiyorum, ancak unutmayın ki Amazon hesabınızın işlem geçmişine erişmek için Amazon Web Services (AWS) kullanmanız gerekebilir. Bu, Amazon'un veri toplama, saklama ve işleme hizmetlerini sunan bir bulut platformudur. AWS ile Amazon verilerinize erişmek için ilk olarak bir AWS hesabı oluşturmanız ve hesabınızı yapılandırmanız gerekir.

yine Python dilinde çalışan bir hesap makinesi örneği veriyorum. Bu örnekte, kullanıcıdan iki sayı alınır ve daha sonra kullanıcının seçtiği işlem (toplama, çıkarma, çarpma veya bölme) için bu iki sayı kullanılarak sonuç hesaplanır.
